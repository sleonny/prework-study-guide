HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is used to create the overall structure of a webpage
  • The head element can not be seen by the user
  • The body element is what is shown to the user
  • Elements are always composed of opening and closing tags
  • Metadata tells the browser how to render the page
  • Metadata includes things like charachter set, keywords, author, viewport and many others

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can write out a color or you can look up the hex# and use that. You can find these on htmlcolorcodes.com
  • A class selector will apply to all elements you assign it to

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • CLI stands for command line interface and I use gitbash for this
  • git pull orgin main is the way to make sure you are in sync between local environment and github
  • git commit -m "______" will allow you to leave a message in code
  • git -b will create a new branch
  • git push orgin pushes work from local environment to github
  • IDE stands for integrated development environment which is a workspace developers use to code for example VS Code

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is used to add interactivity to a website
  • Tools for Java include browser applicatons, 3rd party APIs and 3rd party frameworks and libraries
  • Java is a dynamic and weakly typed language
  • Primitive values define immutable values at the lowest level of the language. Examples are Boolean, Number, String, Symbol, etc.
  • Stings are always surrounded by quotation marks
  • Java will go in the body element of HTML
  • The end of each line will always get a semi-colon
  • An array is a group of related data stored in a single variable
  • Index is a unique number of where one particular thing is in an array
  • When counting in Java it is important to always start with zero, not one
  • A for loop uses predicatable pattern of indexes to perform a task over and over with less coding
  • A for loop must have 3 things, a starting point of the loop, a condition and code that allows the array to iterate over each item in the indexes
  • Function is a set of instructions that tell a computer what to do
  • Functions do not automatically occur, you have to call it by name to execute
  • A function is always followed by ()
